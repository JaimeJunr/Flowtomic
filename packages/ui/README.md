# flowtomic/ui

Componentes UI reutilizáveis baseados em Radix UI e Tailwind CSS.

## Instalação

```bash
bunx flowtomic add button
```

## Uso

```tsx
import { Button } from "flowtomic/ui";

function App() {
  return (
    <Button variant="default" size="md">
      Click me
    </Button>
  );
}
```

## Componentes Disponíveis

- Button
- Card
- Input
- Badge
- Alert
- Dialog
- ... (em desenvolvimento)
